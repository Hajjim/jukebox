Le fichier XML permet de configurer vos préférences.

-------------
Mode d'emploi :
-------------

L’application Jukebox se lance directement
via le fichier ’Exe’ fourni. Néanmoins, il faut
veillé à modifier les paramètres adéquats sur
le fichier de configuration XML apporté, se
trouvant également dans le même répertoire.
Grâce à ce fichier, vous pourrez stipuler le
bon chemin de votre répertoire de musique,
sachant que celui-ci devrait contenir que des
audio au format WAV.

Ainsi, aucun matériel spécifique n’est requis si
ce n’est que le PC doit être connecté à internet.
Une fois le Jukebox lancé, celui-ci joura directement
vos musiques dans l’ordre alphabétique
puisque les compteurs de votes seront nuls.


Par la suite, l’application smartphone devrait
être mise à disposition pour les clients et
ces derniers devront avoir accès au Wi-fi via
leur smartphone.

Après le téléchargement du ClientJukebox,
les clients devront souscrire (dans le menu
de démarrage) l’adresse IP du PC sur-lequel
le Jukebox se trouve, pour pouvoir établir la
connexion.
En étant connecté au serveur, les utilisateurs
recevront la liste des titres avec laquelle ils
devront interagir pour pouvoir choisir leur
chanson favorite !
Pour finir, les clients pourront entrer un pseudo
et voir apparaître leur divers choix sur la barre
d’actualité en haut de l’application.
D’ailleurs, dans le cas où aucune connexion
n’a été établie, alors il s’agira d’un blocage au
niveau du pare-feu et il vous faudra créé une
règle afin que la connexion puisse passer entre
cliens et serveur.
